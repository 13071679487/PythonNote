## 笔记部分

```xml
1、字典(dict)
    1、作用：
        方便数据的检索
    2、概念：
        1、可以存出任意类型的数据的序列
        2、字典是以键值对的方式进行存储
        3、字典是无序的，通过键进行索引取值
        4、键必须是不可变的数据类型，键是唯一的
        5、字典的数据是没有先后顺序的
    3、表示方式：
        {键1：值1,.....}
    4、创建字典
        1、空字典
            {}
            dict(iterable)
        2、非空字典
            1、直接存放数据
                d = {1:'one', 3.14:'pi'}
            2、存储元组、列表
                d2 = dict([(2,4), ('one','m')])
            3、关键字传参
                d3 = dict(a=3,b=5,c=7)
            4、zip函数
                d4 = dict(zip(L1, L2))

    5、CDUR（或CRUD）【增删改查】操作
         1、增加数据
            d5['key'] = len('key')      # {'key': 3}
         2、修改数据
            d5['key'] = 4    # {'key': 4}

            # 如果键存在，进行索引赋值，则是修改数据，否则增加数据
         3、删除数据
            del d5['key']    # {}
         4、查看数据
            in / not in
                'a' in d3       # True

    6、字典常用方法：
        dir(dict)

        dict.clear()        # 清空字典
        dict.copy()         # 字典的拷贝（浅拷贝）
        dict.get(key)       # 通过键查看键所对应的值
        dict.items()        # 返回以列表形式，数据以（键，值）形式表示
        dict.keys()         # 将字典中的所有键以列表的形式呈现
        dict.values()       # 将字典中的所有值以列表的形式呈现
        dict.pop(key)       # 删除字典中键所对应的整条数据
        dict.update(dict1)  # 将一个字典中的数据添加到另一个字典中

    7、字典的遍历
        格式：for 变量 in 非空字典:
             for key, value in 非空字典：

    8、字典推导式
        # [i for i in range(10) if i % 2 ==1]    # 列表推导式
        作用：使用可迭代对象生成一个非空的字典
        格式：{key:value for 变量 in 可迭代对象（iterable）[if 条件表达式]}
            {k:k**2 for k in range(1, 11)}


2、集合(set)
    1、作用：数据去重，减少数据对计算机内存的占用
    2、概念：
        1、可变的序列，集合内的元素是唯一的
        2、无序的存储结构（没有先后顺序）
        3、集合内的元素必须是不可变的数据类型

    3、表达方式：
        也是使用：{}（里面是的元素是单个元素【区别字典】）

    4、创建方式：
        1、空集合
            set()
        2、非空集合
            {2, 23.4}

    5、集合的基本运算：
        交集、并集、补集、子集、超集

        &  生成两个集合的交集,返回交集元素组成的集合
        |  生成两个集合的并集,返回并集元素组成的集合
        -  生成两个集合的补集,返回补集元素组成的集合
        <  判断一个集合是否另一个集合的子集
        >  判断一个集合是否另一个集合的超集

        == 判断两个集合是否相等
        in not in  判断一个元素是否在集合内

    6、常用方法：
        dir(set)

        set.add(元素)     # 向集合中增加一个元素(随机位置)
        set.remove(元素)  # 删除集合中指定元素
        set.pop()        # 从集合的最前面开始删除一个元素
        set.clear()      # 清空集合
        set.copy()       # 拷贝set中的元素
        set.update(set1)    # 将两个集合进行合并

    7、集合推导式
        作用：遍历可迭代对象方式生成一个非空集合
        格式：{表达式 for 变量 in 可迭代对象 [if 条件表达式]}
        例：s = {chr(i) for i in range(97, 123)}


3、固定集合(frozenset)
    1、概念：
        1、不可变的序列
        2、无序的
        3、元素也是唯一的
    2、创建
        1、空集合
            frozenset()
        2、非空集合：
            frozenset(iterable)

            s = frozenset('abc')
    3、固定集合的运算
            集合的运算符否都可以

    4、方法：
        了解


4、函数
    1、作用：实现代码的复用

    2、格式：def 函数名([参数列表])：     # 参数列表 --> 形参
                语句块
                [return 表达式]
        注意：
            1、函数名：变量，符合变量的命名规则
                函数名、变量、类中的方法：名字一般小写（小驼峰命名法）
                类名：一般大写（大驼峰命名法）
            2、参数列表：
                根据执行的内容决定是否定义形参，也可以没有形参
            3、return语句
                1、如果return语句不存在，默认返回为空（None）
                2、如果有return语句，无表达式，则返回为空（None）,否则返回表达式内容
                3、return语句后面的内容不会被执行
            4、pass语句
                空语句

    3、函数的调用：
        函数名([参数列表])     # 参数列表 --> 实参

    4、函数的实参传递方式
        1、位置传参
           实参的值与形参变量按照位置一一对应的传参方式
           例：
                def func(a, b, c):
                    print('%d %d %d' % (a, b, c))

                func(2, 4, 8)


        2、序列传参
            用 * 将序列中元素进行拆解按照位置传参
            例：
                def func(a, b, c):
                    print('%d %d %d' % (a, b, c))

                L = [2, 4, 8]
                func(*L)


        3、关键词传参
            按照实参的变量名对应到形参的变量名，将形参的值赋给形参列表中的变量
            例：
                def func(a, b, c):
                    print('%d %d %d' % (a, b, c))

                func(a=4, b=8, c=2)


        4、字典关键字传参
            用 ** 将字典拆解后按照关键字传参
            例：
                def func(a, b, c):
                    print('%d %d %d' % (a, b, c))

                d = {'a':3, 'b':7, 'c':5}
                func(**d)

        5、优先级
            位置传递、序列传参的优先级高于关键字传参与字典关键字传参。


    5、形参的定义方式
        1、位置形参
            按照实参位置传参的方式接收

        2、星号元组形参
            作用：收集多余的实参
            格式：def 函数名(*args):
            例：
                def func(*args):
                    print(args)

                func(2, 'a', 3, 'b', 4, 5, 6)

        3、命名关键字形参
            作用：限定实参的传递方式必须为关键传参的方式
            格式：def func(*, b, c):
            返回值：元组
            例：
                def func(*, a, b):
                    print(a, b)

                func(b=23, a=43)

        4、双星号形参
            作用:收集多余的关键字传参
            格式：def 函数名(**kwargs):
            返回值：字典
            例：
                def func(**kwargs):
                    print(kwargs)

                func(b=23, a=43, c=45, g=12)

    6、函数的缺省参数
        格式：def 函数名(变量1, 变量2='', ...)
        注意：
            1、如果在形参列表中一个变量是有默认值，那么它的右边的参数必须要有默认值
            2、缺省参数可以有0个或多个
        例：
            def infos(name, age=18, address='None'):
                print('%s 的年龄为：%d,住在：%s' % (name, age, address))

            # infos('wuyi')       # 传递一个实参
            # infos('wuyi', 30)     # 覆盖age的默认值
            infos('wuyi', 30, '广东江门')    # 覆盖age、address的默认值

    7、全局变量与局部变量
        全局变量：定义在函数外的变量
        局部变量：定义在函数内部的变量
        例：
            age = 18        # 全局变量
            def Age():
                age = 22    # 局部变量
                print(f'Age:{age}')

            Age()
            print(age)

    8、global语句
        作用：将局部变量修改为全局变量
        注意：先声明后赋值
        例：
            age = 18        # 全局变量
            def Age():
                global age
                age = 22    # 局部变量
                print(f'Age:{age}')

            Age()
            print(age)

    9、lambda表达式：
        匿名函数（无函数名）
        作用：创建一个匿名的函数对象，不用提供函数名
        格式：lambda 变量1,..:表达式
        例：
            func1 = lambda x:x+2
            print(func1(5))
        优点：创建简单，随时创建，随时销毁
        缺点：只能进行简单的函数功能实现


10、模块
    1、概念：
        由一系列的变量、函数、类等组成的程序组，模块名一般也是以 .py 结尾的文件

    2、分类：
        1、内置的模块
        2、标准库的模块
        3、第三方模块
        4、自己写的模块

    3、模块的导入
        格式：import 模块名 [as 新名]
             from 库名/模块名 import 变量名, 方法名 [as 新名]
             from 库名/模块名 import *    # 导入模块中的全部方法（造成意想不到的错误）
```
